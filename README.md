# LinkedIn Profiles Scrapper

This project uses Selenium 4.9.0 and EdgeDriver for web scraping LinkedIn profiles with an auto-login feature.

## Instructions

1. Change the LinkedIn search query in `vars.py` to match your requirements.

![Screenshot 1](https://github.com/2tieatie/test_task/assets/103947853/203e08b1-b18c-4107-a5d8-be1bd34c35c1)

2. Run `main.py` to start the web scraping process.

![Screenshot 2](https://github.com/2tieatie/test_task/assets/103947853/0a40f40b-dfbd-4c0a-a084-b693f7b055e8)

The EdgeDriver uses options to hasten page loading and simplify HTML (edit it in `vars.py` if you encounter problems).

**WARNING:** A single query retrieves only the first 100 pages of data (due to LinkedIn algorithms).
